Examples
========

Some of examples are trans-compiled from fabianishere/brainfuck/examples[1] at Github

Files
=====

helloworld.rona - prints "Hello World!"
fibo.rona       - prints fibonacci sequence[2] until keyboard interrupt or stack overflow

other BF codes can be easily trans-compiled as well.

1. https://github.com/fabianishere/brainfuck/tree/master/examples
